## How to run?

Create a .env in the backend folder.

.env
```
SECRET_KEY='fitness'

# testing/production
MODE='testing'

PORT=3001

DB_HOST=localhost
DB_NAME=fitness
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=<password>
```

```bash
cd backend
npm install
npm run dev
```

```
cd frontend
npm install
npm start
```
